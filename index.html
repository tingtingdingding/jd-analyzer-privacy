Chrome 扩展隐私政策

声明：我们高度重视您的隐私。本扩展不会收集、存储或传输任何个人身份信息到我们的服务器。

1. 信息收集

JD 智能分析器扩展 不会收集 以下信息：

  • 个人身份信息（姓名、邮箱、电话等）
  • 浏览历史记录
  • Cookies 或其他跟踪数据
  • 用户行为分析数据

2. 本地存储的数据

扩展使用 Chrome Storage API 在您的浏览器本地存储以下信息：

  • API Key：您自行配置的 AI 服务 API 密钥
  • API Endpoint：您选择的 API 服务地址
  • 模型名称：您指定的 AI 模型名称

这些数据仅存储在您的浏览器中，永远不会传输到我们的服务器或任何第三方服务器。

3. 数据传输说明

当您使用扩展分析职位描述时：

  • 您选中的职位描述文本会被发送到您自己配置的 API 服务（如 Groq、OpenAI 等）
  • 数据传输发生在您的浏览器与您选择的 API 服务之间
  • 我们不会作为中间方接收或处理任何数据

4. 权限说明

扩展请求以下权限及其用途：

  • storage：保存您的 API 配置信息
  • activeTab：获取您在当前标签页选中的文本
  • scripting：在页面中执行脚本以提取选中文本
  • host permissions：支持在任意招聘网站使用扩展

5. 第三方服务

本扩展需要您自行配置第三方 AI API 服务（如 Groq、OpenAI 等）。使用这些服务时，请参阅相应服务的隐私政策：

  • Groq 隐私政策
  • OpenAI 隐私政策

6. 儿童隐私

本扩展不面向 13 岁以下儿童。我们不会故意收集儿童的个人信息。

7. 隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，请定期查阅。
